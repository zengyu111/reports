结论

自建 > 外装

五个视频推荐的 Skill 中，四个我们已有等价方案。自建的更透明、更可控、不依赖黑盒更新。

文件驱动 > API 编排

双 Agent 通过 workspace 文件交接，不写一行 API 调用。AGENTS.md 铁律 + 当前状态.md = 最可靠的上下文恢复。

规则外挂 = 无限迭代

执行 Agent 协议是外部 Markdown 文件。今天 10 条，踩坑补 1 条，不碰源码。视频的"后验规则引擎"做不到这一点。
